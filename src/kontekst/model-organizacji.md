# Model organizacji

System zarządzania dostępną treścią publiczną opiera się na podziale funkcji organizacyjnych, który zapewnia spójność decyzji i odpowiedzialność.

Model obejmuje pięć członów:
- doradczy
- kierowniczy
- informacyjny
- wykonawczy
- kontrolny

## Przepływ

doradczy -> kierowniczy -> informacyjny -> wykonawczy -> kontrolny

Brak któregokolwiek elementu powoduje:
- brak decyzji
- brak odpowiedzialności
- chaos w publikacji
